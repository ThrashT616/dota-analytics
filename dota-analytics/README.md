# Dota Analytics

Projeto de Engenharia de Dados utilizando a API OpenDota.

Este projeto foi desenvolvido para estudo prático de Python, PostgreSQL, APIs REST, ETL e versionamento com Git, inspirado no conteúdo produzido pelo Teo Me Why.

## Tecnologias

- Python
- PostgreSQL
- Git
- SQLAlchemy
- Docker

## Objetivos

- Consumir API
- Persistir dados
- Gerar análises
- Criar Dashboard
